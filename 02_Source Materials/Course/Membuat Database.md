## Cara Melakukan
- klik start
- Buka shell
- Run MySQL
    - Run mysql
        
        ```Powershell
        mysql -u root -p
        ```
        
    - Create Database
        
        ```PHP
         CREATE DATABASE mysql_db;
        ```
        
    - Menampilkan Database
        
        ```php
        SHOW DATABASES;
        ```
        
    - Hapus database
        
        ```php
        DROP DATABASE mysql_db;
        ```
        
    - Select database
        
        ```php
        USE mysql_latihan;
        ```
