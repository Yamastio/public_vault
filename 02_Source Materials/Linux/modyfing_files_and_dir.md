## Creating file

### Concatenate dengan cat

```bash
cat > hackingskills
```

- Menambahkan ke baris terakhir tanpa merewrite semua isinya

```bash
cat >> hackingskills
```

- Membaca isi file

```bash
cat hackingskills
```

### Membuat file dengan touch

```bash
touch newfile
```

## Creating Directory

```bash
mkdir newdirectory

```

## Copying Directory

```bash
touch oldfile
cp oldfile /root/newdirectory/newfile
```

## Renaming File

```bash
mv newfile newfile2
```

## Removing file

```bash
rm newfile2
```

## Removing directory

```bash
rmdir newdirectory
```
