## Basic Command Linux

### 1.Melihat direktori saat ini

```bash
pwd
```

### 2. Cek login user

```bash
whoami
```

### 3. Navigasi Filesystem

- Merubah direktori

```bash
cd
cd /home/user
cd /home/user/Documents
```

- List direktori

```bash
ls
ls -l
ls -la
```

### 4. Getting Help

- Gunakan `<command> --help`

```bash
aircrakc-ng --help
```

- Gunakan `<command> -h`

```bash
nmap -h
```

- Gunakan `<command> -?`

```bash
nmap -?
```

### 5. Referensi manual page

```bash
man aircrack-ng
```
